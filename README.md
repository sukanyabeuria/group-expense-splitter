💸 Group Expense Splitter

A web application that helps groups manage shared expenses and automatically split bills between friends.

This project is designed for situations like trips, hostel expenses, events, and shared living where multiple people spend money together.

The system automatically calculates who owes money to whom and provides a clear summary of balances.

1️⃣ Project Overview

Managing group expenses manually can be confusing.

This application helps users:

• Add friends to a group
• Record shared expenses
• Automatically split bills
• Track balances between members

The goal of this project is to simplify group expense management using a clean and easy-to-use interface.

---

2️⃣ Features

✔ Add friends to a group
✔ Record shared expenses
✔ Automatically split bills equally
✔ Track who owes money
✔ Display expense summary

Example:

Trip Expense = ₹3000
Friends = 3

Each person pays = ₹1000

If Rahul paid ₹2000

Sukanya owes Rahul ₹500
Anya owes Rahul ₹500
